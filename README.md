# Gym Center – Responsive Website

Sito Web per una palestra, progettata e sviluppata con particolare attenzione a **layout**, **UX** e **micro-interazioni**.  
Il progetto è pensato come esercizio di front-end “realistico”, con una struttura completa e pronta per essere mostrata in portfolio.

---

## Tech Stack

- **HTML5**
- **CSS3**
  - Design system custom
  - Layout responsive
  - Animazioni e micro-interazioni
- **JavaScript (Vanilla)**
  - Carousel custom
  - Scroll reveal con Intersection Observer
  - Mobile menu custom

---

## Features

- Hero full screen
- Sezione “Perché scegliere noi”
- Sezione corsi con immagini
- Carousel trainer custom (senza librerie)
- Sezione prezzi
- CTA finale
- Footer completo
- Menu mobile full-screen
- Scroll reveal animations
- Responsive design
- Attenzione all’accessibilità (prefers-reduced-motion)

## Note

Il progetto utilizza **JavaScript vanilla** senza framework o librerie esterne, con l’obiettivo di dimostrare:

- padronanza delle basi del front-end
- cura del dettaglio visivo
- attenzione all’esperienza utente
- codice leggibile e manutenibile
